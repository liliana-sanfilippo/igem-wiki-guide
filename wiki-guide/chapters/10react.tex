\section{Setting up your React project}\label{sec:setting-up-your-react-project}

\paragraph{Updating Package Managers:}  If you already have \texttt{npm} or \texttt{yarn} installed, consider
updating them to the latest version to avoid compatibility issues.
You do not need both package managers, one should be sufficient.
If you are unsure, read about the
\href{https://www.geeksforgeeks.org/node-js/difference-between-npm-and-yarn/}{differences} between the managers to
make a decision. \\
Otherwise \textbf{Install Package Mangers:}  \index{Package Managers} \index{npm} \index{Node.js} \index{yarn}
\begin{itemize}
    \item \textbf{node}: Make sure you have Node.js installed.
    You can check by running \texttt{node --version}.
    If it’s not installed, download it from the \href{https://nodejs.org/}{Node.js website}.
    \item \textbf{npm:} It should come bundled with Node.js.
    \item \textbf{yarn}: If you don't already have yarn installed, you can usually do so by running the following
    command: \begin{verbatim}
    npm install --global yarn
    \end{verbatim}
\end{itemize}

\paragraph{Clone your project} if you have not already done so (see \nameref{sec:cloning}).
If you want to use my template, simply download the project from \url{https://github.com/liliana-sanfilippo/template-wiki} as a zip file and move or copy everything (or the desired parts if you know what you are doing) into the root folder of your cloned project.
If you do not know what the root directory is, see here: \url{https://www.geeksforgeeks.org/git/how-to-navigate-to-the-git-root-directory/}.

\paragraph{Install Project Dependencies:} Navigate to your project directory in the terminal and run:
\begin{verbatim}
    yarn install
\end{verbatim}
or
\begin{verbatim}
    npm install
\end{verbatim}

\paragraph{Ensure Compatible Node.js Version:} Make sure your Node.js version is either \texttt{\^\ 8.18.0} or \texttt{$>$=20.0.0} to avoid compatibility issues, with \texttt{18.20.0} being the recommended version in 2024. \index{Node.js}
This is a recommendation for using my template wiki and software and does not apply to all projects.

\section{Learning React}\label{sec:learning-react}
There are already many resources for React so there is no use for me to copy what they say here.
I recommend w3schools again: \url{https://www.w3schools.com/react/react_intro.asp} to learn React basics and Geeks
for Geeks for people wanting to get deeper into React: \url{https://www.geeksforgeeks.org/reactjs/react/}.
\paragraph{If you just want to write texts} and put them on the wiki, please learn about:
\begin{itemize}
    \item JSX: \url{https://www.w3schools.com/react/react_jsx.asp} or \url{https://www.geeksforgeeks.org/reactjs/reactjs-jsx-introduction/}
    \item JSX styling/attributes; \url{https://www.w3schools.com/react/react_jsx_attributes.asp}
\end{itemize}
\paragraph{If you want to create simple components,} please additionally learn at least about:
\begin{itemize}
    \item JSX If statements: \url{https://www.w3schools.com/react/react_jsx_if_statements.asp}
    \item React components: \url{https://www.w3schools.com/react/react_components.asp} or \url{https://www.geeksforgeeks.org/reactjs/reactjs-components/}
    \item React Props: \url{https://www.w3schools.com/react/react_props.asp}, \url{https://www.w3schools.com/react/react_props_destructuring.asp}, \url{https://www.w3schools.com/react/react_props_children.asp}
    \item
\end{itemize}
And ideally as well about:
\begin{itemize}
    \item React Conditional Rendering: \url{https://www.w3schools.com/react/react_conditional_rendering.asp} or \url{https://www.geeksforgeeks.org/reactjs/reactjs-conditional-rendering/}
\end{itemize}
For components that work with data sets (e.g.\ automatic profile rendering for the team page) specifically also:
\begin{itemize}
    \item Mapping \url{https://www.w3schools.com/react/react_lists.asp}
\end{itemize}
If you want to import styling either generally into the App or into specific components: 
\begin{itemize}
    \item \url{https://www.w3schools.com/react/react_css_modules.asp}
\end{itemize}
\paragraph{If you want to alter the template} or create more complex components, please learn about:
\begin{itemize}
    \item React Hooks: \url{https://www.w3schools.com/react/react_hooks.asp}
    \item React Router (if applicable): \url{https://www.w3schools.com/react/react_router.asp}
\end{itemize}
\paragraph{There are many useful libraries and packages} for styling or components.
\begin{itemize}
    \item Styling and components: \url{https://www.geeksforgeeks.org/reactjs/react-bootstrap/}
    \item Styling: \url{https://www.geeksforgeeks.org/css/tailwind-css/}
\end{itemize}
