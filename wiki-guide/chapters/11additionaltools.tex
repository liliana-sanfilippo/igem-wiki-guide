\section{Bootstrap} \label{sec:bootstrap}
\href{https://getbootstrap.com/}{Bootstrap} is a front-end framework that simplifies the process of designing responsive and mobile-first websites.
It provides a collection of pre-designed HTML\index{HTML}, CSS, and JavaScript components, such as buttons, forms,
navigation bars, and modals.\\ \newline
You could use Bootstrap for:
\begin{itemize}
    \item \textbf{Responsive Grid System}: \index{Grid System} \index{Design!Responsive}
    Create a multi-column layout for a product listing page that adjusts to 2 columns on tablets and 4 columns on desktops using Bootstrap's grid classes like \texttt{.col-md-6} and \texttt{.col-lg-3}.

    \item \textbf{Navigation Bar}: \index{Navigation Bar}
    Implement a sticky navigation bar at the top of your web page that collapses into a hamburger menu on mobile devices, using Bootstrap's \texttt{.navbar} and \texttt{.navbar-toggler} classes.

    \item \textbf{Image Carousel}:
    Add an image slider on your homepage to showcase featured products or testimonials, utilizing Bootstrap's carousel component with classes like \texttt{.carousel}, \texttt{.carousel-item}, and \texttt{.active}.

    \item \textbf{Cards for Content Display}:
    Design a series of content cards to present team member profiles, each containing a photo, bio, and social media links, utilizing Bootstrap's card component.

    \item \textbf{Buttons with Icons}:
    Create stylish buttons for social media sharing that include icons, by combining Bootstrap’s button classes with Font Awesome icons.

    \item \textbf{Tooltips for Additional Information}:
    Add tooltips to icons or buttons that provide extra context when hovered over, using Bootstrap's tooltip component for improved user experience.

    \item \textbf{Responsive Tables}:
    Create a responsive data table that allows users to scroll horizontally on smaller screens, using Bootstrap’s table classes like \texttt{.table} and \texttt{.table-responsive}.

    \item \textbf{Footer with Social Links}:
    Design a footer that includes links to your social media profiles with Bootstrap’s utility classes to align items properly.
\end{itemize}
